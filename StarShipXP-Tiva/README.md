#StarShipXP - Tiva LaunchPad Pin Mapping

##Tiva LaunchPad Peripherals

###Onboard RGB LED

1. LED Red    PF_1
2. LED Blue 	PF_2
3. LED Green	PF_3

###Onboard User Switch

4. Switch 1		PF_0
5. Switch 2		PF_4

2. LED Green		PC_4
3. LED Blue 		PC_5

###Sensors
